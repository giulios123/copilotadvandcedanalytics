Implementation Details

📋 Prerequisites

• Azure subscription with appropriate permissions
• Application Insights resource configured for Copilot Studio
• Azure Data Explorer cluster (or use free cluster)
• Data connection between App Insights and Azure Data Explorer

🚀 Deployment

• Import dashboard JSON files into Azure Data Explorer
• Configure data source connection to Application Insights
• Adjust time ranges and filters as needed
• Set up auto-refresh intervals (recommended: 5 minutes)

🔧 Customization

• All KQL queries use corrected syntax (timestamp between, isDesignMode handling)
• Queries handle missing dimensions for new channels
• Thresholds can be adjusted in individual queries
• Additional visualizations can be added using the base queries

📊 Data Freshness

• Application Insights ingestion: Near real-time (2-5 min)
• Azure Data Explorer sync: Configurable (recommend 1-5 min)
• Dashboard refresh: Auto (5 min default)
• Historical data: 90 days retention (configurable)
